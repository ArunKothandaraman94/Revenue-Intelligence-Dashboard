# Revenue Intelligence Dashboard

## Overview

This project analyzes the Global Superstore dataset to evaluate revenue performance, profitability, customer behavior, product performance, and operational efficiency. The objective is to build a Business Intelligence solution that supports executive reporting, KPI monitoring, revenue optimization, and data-driven decision making.

---

## Business Problem

Organizations often struggle to identify:

- Which regions generate the highest revenue
- Which product categories drive profitability
- Which customers contribute the most revenue
- Which products create financial losses
- How discounts impact profitability
- Whether shipping costs affect overall margins

This project addresses these questions through SQL analysis, Python analytics, and interactive dashboard reporting.

## Business Recommendations

1. Prioritize investment in high-performing regions and product categories.
2. Review products with persistent negative profitability.
3. Optimize discounting strategy to protect margins.
4. Reduce logistics costs through shipping mode analysis.
5. Focus retention efforts on top revenue-generating customers.
